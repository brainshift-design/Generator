What's new in Generator

•  added basic text nodes (subscription only for now)

•  added basic shape nodes (subscription only for now)

•  added more data flow nodes like List, Select and Repeat (subscription only for now)

•  more number nodes, like Constant, Distribute and Animate

•  node icons are now shown on all nodes (can be hidden)

•  lists can be connected to some variable inputs to perform the operation on all list items (ex. connecting a list of numbers to an add node will add all the numbers in that list together)

•  added a right-click copy menu to the crash dialog

•  replaced restart checkboxes with buttons

•  added 'Import from file' and 'Save selected to file' for nodes

•  adjusting color controls with arrow keys now supports undo

•  unsupported inputs/outputs are now hidden while connecting

•  wires are not colored according to the type they carry

•  fixed wrong invalid colors assigned to styles

•  fixed Mac shortcuts

•  fixed many UI issues
